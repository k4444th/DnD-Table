# dnd_table

## Project setup

#### Config Setup

```
cp .env .env.local 
```

Then change `VUE_APP_DND_API_ENDPOINT` to the address of the backend.

### Installs dependencies
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
